# style.css 構造ガイド

このファイルは、サイト全体のスタイリングを行うメインのCSSファイルです。外部フレームワークに依存せずVanilla CSSで記述されています。

## 行ごとの概要
- **L1-L4**: CSSファイルのヘッダーコメントと概要。
- **L6-L27**: `:root` 変数宣言。サイト全体のカラーパレット（背景、テキスト、アクセントなど）、スペース（navの高さ等）、トランジションの設定。
- **L29-L76**: リセットCSSおよびベーススタイル。スクロールの滑らかさ、フォント設定（日本語/英語）、リンクのデフォルトスタイル。
- **L78-L93**: アクセシビリティ用スキップリンクのスタイル（`.skip-link`）。
- **L95-L175**: ナビゲーションバーのスタイル（`.site-header`, `.nav-inner`, `.nav-links` 等）。ハンバーガーメニューのアニメーション（`.nav-toggle`）を含む。
- **L177-L210**: 全体的なレイアウトのヘルパー（`.container`, `section`, セクション見出し等の共通デザイン）。
- **L212-L275**: `#hero` セクションのスタイル。ヒーローエリアのグラデーション背景やタイポグラフィー、`.btn`（ボタン）のアニメーションデザイン。
- **L277-L311**: `#about` セクション。プロフィール画像の表示（CSS Gridを活用）とテキスト要素のスタイリング。
- **L313-L397**: `#works` セクション（ディスコグラフィー）。カード型UI（`.work-card`）のグリッドレイアウトやホバーエフェクト。
- **L399-L458**: `#achievements` セクション。タイムライン風レイアウトの左線と丸印などのデザイン。
- **L460-L514**: `#contact` セクションとSNSリンク集のスタイル（Flexboxによるアイコン配置やホバー効果）。
- **L516-L528**: `<footer>` セクションのスタイル。
- **L530-L537**: プロフィール画像固有のスタイル修正。
- **L539-L546**: ワークカード画像固有のスタイル修正。
- **L548-L569**: 言語切り替えボタン（`.lang-toggle`）のスタイル。
- **L571-L617**: レスポンシブデザインのメディアクエリ（タブレット、モバイル向けナビゲーションやカラム変更等のオーバーライド）。
- **L619-L632**: アクセシビリティ機能（アニメーション削減の設定）。
- **L634-L723**: `commission.html` 専用のスタイル。
  - **L639-L685**: 料金表テーブル（`.commission-table`）の詳細デザイン。
  - **L687-L718**: 依頼詳細リスト（`.commission-details`, `.commission-item`）のデザイン。
  - **L720-L723**: CTAセクションのスタイル。
